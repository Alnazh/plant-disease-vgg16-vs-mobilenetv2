Tentang CornLeaf

Penjelasan mengenai aplikasi, fitur, teknologi yang digunakan, dan cara kerja CornLeaf secara singkat.

Apa itu CornLeaf

CornLeaf adalah aplikasi web untuk mendeteksi penyakit pada daun jagung secara otomatis menggunakan computer vision. Pengguna cukup mengunggah foto daun jagung, dan aplikasi akan menampilkan hasil klasifikasi dari dua model machine learning sekaligus, yaitu VGG16 dan MobileNetV2, lengkap dengan tingkat keyakinan dan rekomendasi tindakan.

Fitur Utama

  • Prediksi gambar daun jagung secara real-time dari dua model sekaligus
  • Rekomendasi tindakan penanganan berdasarkan hasil klasifikasi
  • Dashboard evaluasi lengkap dengan kurva training dan confusion matrix
  • Penjelasan otomatis di setiap grafik agar mudah dipahami
  • Eksplorasi dataset beserta rincian pembagian data latih dan validasi
  • Tampilan responsif yang dapat diakses dari perangkat mobile

Teknologi yang Digunakan

Py

Flask (Python)

Backend server yang menangani routing halaman, proses inferensi, dan API data evaluasi.

TF

TensorFlow & Keras

Framework untuk membangun dan menjalankan model VGG16 serta MobileNetV2 hasil transfer learning.

Np

NumPy

Mengubah gambar menjadi array numerik yang siap diproses oleh model.

JS

JavaScript Native

Menangani interaksi unggah gambar, pemanggilan API prediksi, dan pembaruan tampilan hasil.

Cj

Chart.js

Menggambar kurva akurasi, kurva loss, dan grafik perbandingan metrik pada halaman Evaluasi.

Sk

Scikit-learn

Menghitung classification report dan confusion matrix saat proses evaluasi model dijalankan.

Cs

HTML & CSS Murni

Tampilan antarmuka dibangun tanpa framework CSS, dengan satu berkas gaya terpusat.

Cara Kerja

1

Unggah Gambar

Pengguna mengunggah foto daun jagung melalui halaman Prediksi.

2

Pra-pemrosesan

Gambar diubah ukurannya menjadi 224x224 piksel dan dikonversi menjadi array numerik menggunakan NumPy dan util Keras.

3

Preprocessing Khusus Model

Array diproses ulang sesuai kebutuhan masing-masing arsitektur, memakai fungsi preprocess_input milik VGG16 dan MobileNetV2 secara terpisah.

4

Inferensi Dua Model

Gambar yang sama diproses oleh VGG16 dan MobileNetV2 secara terpisah untuk menghasilkan probabilitas tiap kelas.

5

Tampilkan Hasil

Kelas dengan probabilitas tertinggi ditampilkan sebagai prediksi, disertai rekomendasi tindakan dan tingkat kesepakatan kedua model.

Aplikasi ini bersifat alat bantu diagnosis awal dan tidak menggantikan pemeriksaan oleh ahli pertanian atau petugas penyuluh lapangan, terutama untuk kasus yang gejalanya tidak jelas atau kedua model memberi hasil yang berbeda.

Karena tidak ada data uji terpisah, seluruh angka akurasi yang ditampilkan di halaman Evaluasi berasal dari data validasi, bukan dari data yang benar-benar belum pernah dilihat model. Untuk menilai performa pada gambar baru, gunakan halaman Prediksi secara langsung.
